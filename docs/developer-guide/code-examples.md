# Code Examples

## Overview

This section provides ready-to-use code examples and automation recipes that demonstrate how to interact with your VergeOS environment programmatically. Whether you're writing quick scripts or building production tooling, these examples give you a practical starting point.

Examples cover common tasks like:

- **VM lifecycle management** — Create, start, stop, snapshot, and delete virtual machines
- **Tenant provisioning** — Automate tenant creation, resource allocation, and onboarding workflows
- **Storage operations** — Manage NAS volumes, shares, and snapshot schedules
- **Monitoring and reporting** — Query system health, collect metrics, and generate usage reports
- **Bulk operations** — Perform batch updates across VMs, networks, or tenants

## Languages

Code examples are provided in several popular languages:

- **Python** — Using the `requests` library and the VergeOS API
- **Go** — Native HTTP client examples for compiled tooling
- **PowerShell** — For Windows-based automation and scripting environments
